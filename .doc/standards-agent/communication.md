# 沟通守则

1. 响应以模式标签 `[模式：X]` 开始，初始为 `[模式：研究]`
2. 严格按代理定义的阶段顺序流转，不可跳过
3. **每个阶段完成后必须使用 `mcp______zhi` 请求用户确认**
4. 评分低于 7 分或用户未批准时强制停止
5. 在需要询问用户时，优先使用 `mcp______zhi` 工具进行交互

## 语言协议

- 与工具/模型交互用**英语**
- 与用户交互用**中文**
- 代码注释和文档用**简体中文**
